# ToDo List

Веб-додаток «Список завдань»

Можливості:

1. **Додавати** завдання.
2. **Відмічати** завдання як виконані.
3. **Видаляти** завдання.

Додатково: фільтри (усі / активні / виконані), індикатор прогресу,
лічильник завдань та очищення виконаних

## Технології

- **Frontend:** HTML, CSS, ванільний JavaScript (без фреймворків)
- **Backend:** Node.js + Express, REST API
- **База даних:** SQLite (бібліотека `better-sqlite3`), файл `todo.db`
  створюється автоматично біля сервера

## Структура

```
kurs_ai/
├── server.js           Express-сервер та REST API
├── db.js               підключення до БД SQLite і запити
├── package.json
├── todo.db             база даних (створюється під час першого запуску)
└── public/             клієнтська частина
    ├── index.html
    ├── style.css
    └── app.js
```

## Запуск

Потрібен [Node.js](https://nodejs.org/) версії 18 або новіше.

```bash
npm install
npm start
```

Після запуску відкрийте у браузері: <http://localhost:3000>

## REST API

| Метод    | Шлях             | Опис                                   |
|----------|------------------|----------------------------------------|
| `GET`    | `/api/tasks`     | Отримати список усіх завдань           |
| `POST`   | `/api/tasks`     | Додати завдання `{ "title": "..." }`   |
| `PATCH`  | `/api/tasks/:id` | Оновити `{ "done": true }` або `title` |
| `DELETE` | `/api/tasks/:id` | Видалити завдання                      |

Приклад завдання:

```json
{
  "id": 1,
  "title": "Зробити лабораторну роботу",
  "done": false,
  "createdAt": "2026-06-19 10:00:00"
}
```
